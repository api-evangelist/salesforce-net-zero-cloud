# Salesforce Net Zero Cloud API (salesforce-net-zero-cloud)
The Salesforce Net Zero Cloud API enables organizations to track, analyze, and report  on their carbon emissions and sustainability data. It provides programmatic access to  environmental data, carbon accounting, and ESG reporting capabilities.


**URL:** [Visit APIs.json URL](https://www.salesforce.com/products/net-zero-cloud/overview/)
